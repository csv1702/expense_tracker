Project: OneStack Expense Tracker
Built with: CodeIgniter 3, Bootstrap 5, MySQL

INSTALLATION STEPS:
1. Copy the 'expense_tracker' folder to your XAMPP htdocs folder.
2. Import 'database_dump.sql' into phpMyAdmin (Database name: expense_tracker).
3. Open application/config/database.php and update hostname/username/password if needed.
   (Currently set to 'localhost:3307' / root / empty).
4. Access via: http://localhost/expense_tracker/
